## python venv

```shell
# 1. 创建
py -m venv ll_env

# 2. 激活/启用 (window)
./ll_env/Scripts/activate

# 3. 退出
deactivate
```

## python requirements

```shell
# （已安装库）生成 requriements.txt
pip freeze > requriements.txt

# 根据 requriements.txt 安装库
pip install -r requriements.txt
```
